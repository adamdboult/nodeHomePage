
\subsection{Monomorphism}

A morphism which is injective. That is:

\(f(a)=f(b)\rightarrow a=b\)

The following are monomorphisms:

\begin{itemize}
\item \(f(x)=x\)
\item \(f(x)=cx\)
\item Converting natural numbers to integers
\end{itemize}

The following are not monomorphisms:

\begin{itemize}
\item \(f(x)=0\)
\item \(f(x)=x+1\)
\end{itemize}
