
\subsection{Org mode}

customising using conf.org file



org mode
use of *
use of +
use of TODO: set with:

\begin{verbatim}
C-c C-t
\end{verbatim}
see agenda
files to be in agenda stored in org-agenda-files

to change state and record note:
\begin{verbatim}
C-u C-c C-t
\end{verbatim}

cycle through agenda files:
\begin{verbatim}
C-'
\end{verbatim}

add current file to agenda files:
\begin{verbatim}
C-c [
\end{verbatim}

remove:
\begin{verbatim}
C-c ]
\end{verbatim}

view agenda:
\begin{verbatim}
M-x org-agenda
\end{verbatim}

change theme:
\begin{verbatim}
M-x load-theme
\end{verbatim}
