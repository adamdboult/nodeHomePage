
\subsection{Covers}

A space \(X\) is covered by a set of subsets of \(X\), \(C\), if the union of \(C\) is \(X\).

\subsection{Subcover}

A subset of \(C\) which still covers \(X\) is a a subcover.

\subsection{Open cover}

\(C\) is an open cover if each member is an open set.
