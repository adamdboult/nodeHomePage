0.1 Complements and disjoint sets

0.1.1 Disjoint sets

Sets are disjoint is there is no overlap in their elements. Two sets are si and sj are mutually exclusive if:

si ∧ sj = ∅

A collection of events s are all mutually exclusive if all pairs are mutually exclusive. That is:

∀si ∈ s∀sj ∈ s[si ∧ sj ≠ ∅ → si = sj]

0.1.2 Complement function

xC is the completement. It is defined such that:

∀x[x ∧ xC = ⌀]

For a set b, the complement with respect to a is all elements in a which are not in b.

∀x ∈ a∀y ∈ b[x ∈ (a \ b)∧y ∈ (a \ b)]

b ∧ (a \ b)=⌀

That is, b and a \ b are disjoint.

0.1.3 Existence of the complement

For two sets a and b we can write (a \ b). This is the set of elements of a which are not in b.

Consider the axiom of specification:

∀x∀a∃s[(P(x)∧x ∈ a)↔(x ∈ s)]

We can also write

∀x∀a∀b∃s[(x ∉ b ∧ x ∈ a)↔(x ∈ s)]

Which provides the complement, s.
