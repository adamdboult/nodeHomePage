1.1 Partial differentiation

1.1.1 The partial differential operator

1.1.1.1 Differential

When we change the value of an input to a function, we also change the output. We can examine these changes.

Consider the value of a function f(x) at points x1 and x2.

y1 = f(x1)

y2 = f(x2)

y2 − y1 = f(x2)−f(x1)

$\frac{y_2-y_1}{x_2-x_1}=\frac{f(x_2)-f(x_1)}{x_2-x_1}$

Let’s define x2 in terms of its distance from x_1:

x2 = x1 + ϵ

$\frac{y_2-y_1}{\epsilon }=\frac{f(x_1+\epsilon )-f(x_1)}{\epsilon }$

We define the differential of a function as:

$\frac{\delta y}{\delta x}=\lim_{\epsilon \rightarrow 0^+}\frac{f(x+\epsilon )-f(x)}{\epsilon }$

If this is defined, then we say the function is differentiable at that point.

1.3.1.2 Defining e

We know that:

$(a+b)^n=\sum^n_{i=0} a^i b^{n-i} \frac{n!}{i!(n-i)!}$

Let’s set b = 1

$(a+1)^n=\sum^n_{i=0} a^i \frac{n!}{i!(n-i)!}$

Let’s set $a=\frac{1}{n}$

$(1+\frac{1}{n})^{n}=\sum^{n}_{i=0} \frac{1}{n^i} \frac{n!}{i!(n-i)!}$

$(1+\frac{1}{n})^{n}=\sum^{n}_{i=0} \frac{1}{i!} \frac{n!}{n^i(n-i)!}$

$\lim_{n\rightarrow \infty }(1+\frac{1}{n})^{n}=\lim_{n\rightarrow \infty }\sum^{n}_{i=0} \frac{1}{i!} \frac{n!}{n^i(n-i)!}$

From the lemma above:

$\lim_{n\rightarrow \infty }(1+\frac{1}{n})^{n}=\sum^{\infty }_{i=0} \frac{1}{i!}$

$e=\sum^{\infty }_{i=0} \frac{1}{i!}$
