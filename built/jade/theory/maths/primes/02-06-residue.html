0.1 Residue systems

0.1.1 Least residue system modulo n

This is the set of numbers from 0 to n − 1.

0.1.2 Complete residue system

This a set of numbers none of which are congruent mod n. That is, for no pair {a, b} does amod(n)=bmod(n)

0.1.3 Reduced residue system

This is a complete residue system where all numbers are relatively prime to n.
