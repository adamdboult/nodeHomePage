0.1 Logarithms

0.1.1 Logarithms

If:

c = ab

Then

logac = b

Product rule:

a = clogca

b = clogcb

So:

ab = clogcab

But also:

ab = clogcaclogcb

ab = clogca + logcb

So:

logca + logcb = logcab

0.1.2 Power rule

a = blogba

So:

ac = blogbac

And separately:

ac = (blogba)c

ac = (bclogba)

So:

clogba = logbac
