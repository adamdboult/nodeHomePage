1 Set theory

1.1 Introduction to sets

1.1.1 Membership relation

Say we have a preterite P(x) which is true for some values of x. Sets allow us to explore the properties of these values.

We may want to talk about a collection of terms for which P(x) is true, which we call a set.

To do this we need to introduce new axioms, however first we can add (conservative) definitions to help us do this.

We introduce a new relation: membership. If element x is in set s then the following relation is true, otherwise it is false:

x ∈ s

Sets are also terms. In first-order logic they will be included in quantifiers. Indeed, in set theory, we aim to treat everything as sets.

If a term is not a member of another term, we can write this using the non-member relation as follows:

∀x∀s[¬(x ∈ s)↔x ∉ s]

1.1.2 Axiom of extensionality

If two sets contain the same elements, they are equal.

∀x∀y[∀z(z ∈ x ↔ z ∈ y)→x = y]

This is an axiom, not a definition, because equality was defined as part of first-order logic.

Note that this is not bidirectional. x = y does not imply that x and y contain the same elements. This is appropraite as $\frac{1}{2}= \frac{2}{4}$ for example, even though they are written differently as sets.

1.1.2.1 Reflexivity of equality

The reflexive property is:

∀x(x = x)

We can replace the instance of y with x:

∀x[∀z(z ∈ x ↔ z ∈ x)→x = x]

We can show that the following is true:

∀z(z ∈ x ↔ z ∈ x)

Therefore:

∀x[T → x = x]

x = x

1.1.2.2 Symmetry of equality

The symmetry property is:

∀x∀y[(x = y)↔(y = x)]

We know that the following are true:

∀x∀y[∀z(z ∈ x ↔ z ∈ y)→x = y]

∀x∀y[∀z(z ∈ x ↔ z ∈ y)→y = x]

So:

∀x∀y[∀z(z ∈ x ↔ z ∈ y)→(x = y ∧ y = x)]

1.1.2.3 Transitivity of equality

The transitive property is:

∀x∀y∀z[(x = y ∧ y = z)→x = z]

1.1.2.4 Substitution for functions

The substitutive property for functions is:

∀x∀y[(x = y)→(f(x)=f(y))]

1.1.2.5 Substitution for formulae

The substitutive property for formulae is:

1.2 Defining sets

1.2.1 Axiom schema of specification

1.2.1.1 The axiom schema of unrestricted comprehension

We want to formalise the relationship between the preterite and the set. An obvious way of doing this is to add an axiom for each preterite in our structure that:

∀x∃s[P(x)↔(x ∈ s)]

This is known as “unrestricted comprehension” and there are problems with this approach.

Consider a predicate for all terms which are not members of themselves. That is:

¬(x ∈ x)

This implies the following is true:

∀x∃s[¬(x ∈ x)↔(x ∈ s)]

As this is true for all x, it is true for x = s. So:

∃s[¬(s ∈ s)↔(s ∈ s)]

This statement is false. As we have inferred a false formula, the axiom of unrestricted comprehension does not work. This result is known as Russel’s Paradox.

This is an axiom schema rather than an axiom. That is, there is a new axiom for each preterite.

1.2.1.2 Axiom schema of specification

To resolve Russels’ paradox, we amend the axiom schema to:

∀x∀a∃s[(P(x)∧x ∈ a)↔(x ∈ s)]

That is, for every set a, we can define a subset s for each predicate.

This resolves Russel’s Paradox. Let’s take the same steps on the above formula as in unrestricted comprehension;

∀x∀a∃s[(¬(x ∈ x)∧x ∈ a)↔(x ∈ s)]

∃s[(¬(s ∈ s)∧s ∈ s)↔(s ∈ s)]

So long as the subsets s are not members of themselves, this holds.

1.2.7 Cardinality

1.2.7.1 Cardinality of finite sets

The cardinality of a set s is shown as |s|. It is the number of elements in the set. We define it formally below.

1.2.7.2 Injectives, surjectives and bijectives

Consider 2 sets. If there is an injective from a to b then for every element in a there is a unique element in b.

If this injective exists then we say |a|≤|b|.

Similarly, if there is a surjective, that is for every element in b there is a unique element in a, then |a|≥|b|.

Therefore, if there is a bijection, |a|=|b|, and if there is only an injective or a surjective then |a|<|b| or |a|>|b| respectively.

1.2.7.3 Cardinality as a function

Every set has a cardinality. As a result cardinality cannot be a well-defined function, for the same reason there is no set of all sets.

Cardinality functions can be defined on subsets.

1.3 Ordering

1.3.1 Inequalities

1.3.1.1 Less than or equal

Orderings define relations between elements in a set, where one element can preceed the other.

Orderings are antisymmetric. That is, the only case where the relation is satisfied in both directions is if the elements are equal.

(a ≤ b)∧(b ≤ a)→(a = b)

Orderings are transitive. That is:

(a ≤ b)∧(b ≤ c)→(a ≤ c)

1.3.1.2 Greater than or equal

1.3.1.3 Less than and greater than

The relation ≤ is refered to as non-strict.

There is a similar strict relation relation, <:

(a ≤ b)∧¬(b ≤ a)→(a < b)

1.3.2 Ordered sets

1.3.2.1 Totally ordered sets

A totally ordered set is one where the relation is defined on all pairs:

∀a∀b(a ≤ b)∨(b ≤ a)

Note that totality implies reflexivity.

1.3.2.2 Partially ordered sets (poset)

A partially ordered set, or poset, is one where the relation is defined between each element and itself.

∀a(a ≤ a)

That is, every element is related to itself.

These are also called posets.

1.3.2.3 Well-ordering

A well-ordering on a set is a total order on the set where the set contains a minimum number. For example the relation ≤ on the natural numbers is a well-ordering because 0 is the minimum.

The relation ≤ on the integers however is not a well-ordering, as there is no minimum number in the set.

1.3.3 Intervals

For a totally ordered set we can define a subset as being all elements with a relationship to a number. For example:

[a, b]={x : a ≤ x ∧ x ≤ b}

This denotes a closed interval. Using the definition above we can also define an open interval:

(a, b)={x : a < x ∧ x < b}

1.3.4 Infinitum and supremum

1.3.4.1 Infinitum

Consider a subset S of a partially ordered set T.

The infinitum of S is the greatest element in T that is less than or equal to all elements in S.

For example:

inf[0, 1]=0

inf(0, 1)=0

1.3.4.2 Supremum

The supremum is the opposite: the smallest element in T which is greater than or equal to all elements in S.

sup[0, 1]=1

sup(0, 1)=1

1.3.4.3 Max and min

If the infinitum of a set S is in S, then the infinimum is the minimum of set S. Otherwise, the minimum is not defined.

min[0, 1]=0

min(0, 1) isn’t defined.

Similarly:

max[0, 1]=1

max(0, 1) isn’t defined.

1.4 Set algebra

1.4.1 Set union and intersection

We discuss functions. Just because we can write a function of sets which exist, does not mean the results of the functions exist. For that we need axioms discussed later.

1.4.1.1 Union function

We define a function on two sets, a ∨ b, such that the result contains all elements from either sets.

∀a∀x∀y[a ∈ (x ∨ y)↔(a ∈ x ∨ a ∈ y)]

This is commutative: a ∨ b = b ∨ a

This is associative: (a ∨ b)∨c = a ∨ (b ∨ c)

1.4.1.2 Intersection function

We define a function, a ∧ b, on two sets, such that the result contains all elements which are in both.

∀a∀x∀y[a ∈ (x ∧ y)↔(a ∈ x ∧ a ∈ y)]

This is commutative: a ∧ b = b ∧ a

This is associative: (a ∧ b)∧c = a ∨ (b ∧ c)

1.4.1.3 Distribution of union and intersection

Union is distributive over intersection: a ∨ (b ∧ c)=(a ∨ b)∧(a ∨ c)

Intersection is distributive over union: a ∧ (b ∨ c)=(a ∧ b)∨(a ∧ c)

1.4.2 Complements and disjoint sets

1.4.2.1 Disjoint sets

Sets are disjoint is there is no overlap in their elements. Two sets are si and sj are mutually exclusive if:

si ∧ sj = ∅

A collection of events s are all mutually exclusive if all pairs are mutually exclusive. That is:

∀si ∈ s∀sj ∈ s[si ∧ sj ≠ ∅ → si = sj]

1.4.2.2 Complement function

xC is the completement. It is defined such that:

∀x[x ∧ xC = ⌀]

For a set b, the complement with respect to a is all elements in a which are not in b.

∀x ∈ a∀y ∈ b[x ∈ (a \ b)∧y ∈ (a \ b)]

b ∧ (a \ b)=⌀

That is, b and a \ b are disjoint.

1.4.2.3 Existence of the complement

For two sets a and b we can write (a \ b). This is the set of elements of a which are not in b.

Consider the axiom of specification:

∀x∀a∃s[(P(x)∧x ∈ a)↔(x ∈ s)]

We can also write

∀x∀a∀b∃s[(x ∉ b ∧ x ∈ a)↔(x ∈ s)]

Which provides the complement, s.

1.4.3 Axiom of union

1.4.3.1 Motivation

While we have described various sets, we have not said that they exist. That is, if A and B both exist, then currently we cannot ensure A ∧ B exists, just that it can be described.

The axiom of union enables us to ensure all sets from unions and intersections exist.

1.4.3.2 Axiom of union

∀a∃b∀c[c ∈ b ↔ ∃d(c ∈ d ∧ d ∈ a)]

That is, for every set a, there exists a set b where all the elements in b are the elements of the elements in a.

Here, b is the union of the sets in a.

1.4.4 Boolean algebra

1.4.4.1 Boolean algebra in propositional logic

We previously discussed properties of normal form, and the results from these properties.

If another structure shares these properties then they will also share the results.

1.4.4.2 Sets satisfy the definitions of a boolean algebra

If a mathematical structure has the following properties, it shares the results from normal form, and is a boolean algebra.

  • Both binary operators are commutitive - A ∧ B = B ∧ A and A ∨ B = B ∨ A

  • Both binary operators are associative - (A ∧ B)∧C = A ∧ (B ∧ C) and (A ∨ B)∨C = A ∨ (B ∨ C)

  • Completements - A ∧ ¬A = ∅ and A ∨ ¬A = U

  • Absorption - A ∧ (A ∨ B)=A and A ∨ (A ∧ B)=A

  • Identity - A ∧ U = A and A ∨ ∅ = A

  • Distributivity - A ∧ (B ∨ C)=(A ∧ B)∨(A ∧ C) and A ∨ (B ∧ C)=(A ∨ B)∧(A ∨ C)

These hold for sets, and so boolean algebra holds for sets.

1.4.5 Algebra on a set

1.4.5.1 Standard algebra

An algebra, Σ, on set s is a set of subsets of s such that:

  • Closed under intersection: If a and b are in Σ then a ∧ b must also be in Σ

  • ∀ab[(a ∈ Σ ∧ b ∈ Σ)→(a ∧ b ∈ Σ)]

  • Closed under union: If a and b are in Σ then a ∨ b must also be in Σ.

  • ∀ab[(a ∈ Σ ∧ b ∈ Σ)→(a ∨ b ∈ Σ)]

If both of these are true, then the following is also true:

  • Closed under complement: If a is in Σ then s ∖ a must also be in Σ

We also require that the null set (and therefore the original set, null’s complement) is part of the algebra.

1.6.3 Cantors theorem

The cardinality of the powerset is strictly greater than the cardinality of the underlying set.

That is, |P(s)| < |s|.

This applies to finite sets and infinite sets. In particular, this means that the powerset of the natural numbers is bigger than the natural numbers.

1.6.3.1 Proof

If one set is at least as big as another, then then is a surjection from that set to the other.

That is, if we can prove that there is no surjection from a set to its powerset, then we have proved the theorem.

We consider f(s). If there is a surjection, then for every subset of s there should be a mapping from s to that subset.

We take set s and have the powerset of this, P(s).

Consider the set:

A = {x ∈ s|x ∉ f(x)}

That is, the set of all elements of s which do not map to the surjection.
