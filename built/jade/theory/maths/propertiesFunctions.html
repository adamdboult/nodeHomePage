Defining odd and even functions

An even function is one where:

f(x)=f(−x)

An odd function is one where:

f(x)= − f(−x)

Functions which are even and odd

If a function is even and odd:

f(x)=f(−x)= − f(−x)

f(x)= − f(x)

Then f(x)=0.

Concave and convex functions

Convex functions

A convex function is one where:

∀x1, x2 ∈ ℝ∀t ∈ [0, 1][f(tx1 + (1 − t)x2 ≤ tf(x1)+(1 − t)f(x2)]

That is, for any two points of a function, a line between the two points is above the curve.

A function is strictly convex if the line between two points is strictly above the curve:

∀x1, x2 ∈ ℝ∀t ∈ (0, 1)[f(tx1 + (1 − t)x2 < tf(x1)+(1 − t)f(x2)]

An example is y = x2.

Concave functions

A concave function is an upside down convex function. The line between two points is below the curve.

∀x1, x2 ∈ ℝ∀t ∈ [0, 1][f(tx1 + (1 − t)x2 ≥ tf(x1)+(1 − t)f(x2)]

A function is strictly concave if the line between two points is strictly below the curve:

∀x1, x2 ∈ ℝ∀t ∈ (0, 1)[f(tx1 + (1 − t)x2 > tf(x1)+(1 − t)f(x2)]

An example is y = −x2.

Affine functions

If a function is both concave and convex, then the line between two points must be the function itself. This means the function is an affine function.

Continous functions

A function is continuous if:

limx → cf(x)=f(c)

For example a function $\frac{1}{x}$ is not continuous as the limit towards 0 is negative infinity. A function like y = x is continous.

More strictly, for any ϵ > 0 there exists

δ > 0

c − δ < x < c + δ

Such that

f(c)−ϵ < f(x)<f(c)+ϵ

This means that our function is continuous at our limit c, if for any tiny range around f(c), that is f(c)−ϵ and f(c)+ϵ, there is a range around c, that is c − δ and c + δ such that all the value of f(x) at all of these points is within the other range.

Boundedness theorem

If f(x) is closed and continuous in [a, b] then f(x) is bounded by m and M. That is:

∃m ∈ ℝ∃M ∈ ℝ∀x ∈ [a, b](m < f(x)<M

Intermediate value theorem

Take a real function f(x) on closed interval [a, b], continuous on [a, b, ].

IVT says that for all numbers u between f(a) and f(b), there is a corresponding value c in [a, b] such that f(c)=u.

That is:

∀u ∈ [min(f(a),f(b)), max(f(a),f(b))]∃c ∈ [a, b](f(c)=u)

Extreme value theorem

We can expand the boundedness theorem such that m and M are functions of f(x) in the bound [a, b]. That is:

∃m ∈ ℝ∃M ∈ ℝ∀x ∈ [a, b](m < f(x)<M)
