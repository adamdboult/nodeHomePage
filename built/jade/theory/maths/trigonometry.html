Sine and cosine

Defing sine and cosine using Euler’s formula

Euler’s formula

Previously we showed that:

$e^x=\sum_{i=0}^\infty \frac{x^i}{i!}$

Consider:

eiθ

$e^{i\theta }=\sum_{j=0}^\infty \frac{(i\theta )^j}{j!}$

$e^{i\theta }=[\sum_{j=0}^\infty \frac{(\theta )^{4j}}{(4j)!}-\sum_{j=0}^\infty \frac{(\theta )^{4j+2}}{(4j+2)!}]+i[\sum_{j=0}^\infty \frac{(\theta )^{4j+1}}{(4j+1)!}-\sum_{j=0}^\infty \frac{(\theta )^{4j+3}}{(4j+3)!}]$

We then use this to define sin  and cos  functions.

$\cos (\theta ):=\sum_{j=0}^\infty \frac{(\theta )^{4j}}{(4j)!}-\sum_{j=0}^\infty \frac{(\theta )^{4j+2}}{(4j+2)!}$

$\sin (\theta ):=\sum_{j=0}^\infty \frac{(\theta )^{4j+1}}{(4j+1)!}-\sum_{j=0}^\infty \frac{(\theta )^{4j+3}}{(4j+3)!}$

So:

eiθ = cos(θ)+isin(θ)

Alternative formulae for sine and cosine

We know

eiθ = cos(θ)+isin(θ)

e−iθ = cos(θ)−isin(θ)

So

eiθ + e−iθ = cos(θ)+isin(θ)+cos(θ)−isin(θ)

$\cos (\theta )=\frac{e^{i\theta }+e^{-i\theta }}{2}$

And

eiθ − e−iθ = cos(θ)+isin(θ)−cos(θ)+isin(θ)

$\sin (\theta )=\frac{e^{i\theta }-e^{-i\theta }}{2i}$

Sine and cosine are odd and even functions

Sine is an odd function.

sin(−θ)= − sin(θ)

Cosine is an even function.

cos(−θ)=cos(θ)

Addition of sine and cosine

Adding waves with same frequency

We know that:

asin(bx + c)=asin(bx)cos(c)+asin(c)cos(bx)

So:

asin(bx + c)+dsin(bx + e)=asin(bx)cos(c)+asin(c)cos(bx)+dsin(bx)cos(e)+dsin(e)cos(bx)

We know that:

$\sin(\theta )=\frac{e^{i\theta }-e^{-i\theta }}{2i}$

So:

$a\sin(bx+c)+d\sin(bx+f)=a\frac{e^{i(bx+c)}-e^{-i(bx+c)}}{2i}+d\frac{e^{i(bx+f)}-e^{-i(bx+f)}}{2i}$

$a\sin(bx+c)+d\sin(bx+f)=\frac{a(e^{i(bx+c)}-e^{-i(bx+c)})+d(e^{i(bx+f)}-e^{-i(bx+f)})}{2i}$

$a\sin(bx+c)+d\sin(bx+f)=\frac{a(e^{ibx}e^{ic}-e^{-ibx}e^{-ic})+d(e^{ibx}e^{if}-e^{-ibx}e^{-if)})}{2i}$

$a\sin(bx+c)+d\sin(bx+f)=\frac{(e^{ibx}(ae^{ic}+de^{if})-e^{-ibx}(ae^{-c}+d^{-if})}{2i}$

aisin(bix + ci)+ajsin(bjx + cj)=aisin(bix + ci)+ajsin(bix + cj)

aisin(bix + ci)+ajsin(bjx + cj)=aisin(bix)cos(ci)+aisin(ci)cos(bix)+ajsin(bix)cos(cj)+ajsin(cj)cos(bix)

Calculus of sine and cosine

Unity

Note that with imaginary numbers we can reverse all is. So:

eiθ = cos(θ)+isin(θ)

e−iθ = cos(θ)−isin(θ)

eiθe−iθ = (cos(θ)+isin(θ))(cos(θ)−isin(θ))

eiθe−iθ = cos(θ)2 + sin(θ)2

eiθe−iθ = eiθ − iθ = e0 = 1

So:

cos(θ)2 + sin(θ)2 = 1

Note that if cos(θ)2 = 0, then sin(θ)2 = ±1

That is, if the real part of eiθ is 0, the imaginary part is ±1. And visa versa.

Similarly if the derivative of the real part of eiθ is 0, the imaginary part is ±1. And visa versa.

Sine and cosine are linked by their derivatives

Note that these functions are linked in their derivatives.

$\frac{\delta }{\delta \theta }\cos (\theta )=\sum_{j=0}^\infty \frac{(\theta )^{(4j+3)}}{(4j+3)!}-\sum_{j=0}^\infty \frac{(\theta )^{4j+1}}{(4j+1)!}$

$\frac{\delta }{\delta \theta }\cos (\theta )=-\sin (\theta )$

Similarly:

$\frac{\delta }{\delta \theta }\sin (\theta )=cos(\theta )$

Both sine and cosine oscillate

$\frac{\delta^2 }{\delta \theta^2}\sin (\theta )=-\sin (\theta )$

$\frac{\delta^2 }{\delta \theta^2}\cos (\theta )=-\cos (\theta )$

So for either of:

y = cos(θ)

y = sin(θ)

We know that

$\frac{\delta^2 }{\delta \theta^2}y(\theta )=-y(\theta )$

Consider θ = 0.

ei.0 = cos(0)+isin(0)

1 = cos(0)+isin(0)

sin(0)=0

cos(0)=1

Similarly we know that the derivative:

sin′(0)=cos(0)=1

cos′(0)= − sin(0)=0

Consider cos(θ).

As cos(0) is static at θ = 0, and is positive, it will fall until cos(θ)=0.

While this is happening, sin(θ) is increasing. As:

cos(θ)2 + sin(θ)2 = 1

sin(θ) will equal 1 where cos(θ)=0.

Due to symmetry this will repeat 4 times.

Let’s call the length of this period τ.

Where θ = τ * 0

  • cos(θ)=1

  • sin(θ)=0

Where $\theta =\tau *\frac{1}{4}$

  • cos(θ)=0

  • sin(θ)=1

Where $\theta =\tau *\frac{2}{4}$

  • cos(θ)= − 1

  • sin(θ)=0

Where $\theta =\tau *\frac{3}{4}$

  • cos(θ)=0

  • sin(θ)= − 1

Relationship betweencos(θ) and sin(θ)

Note that $\sin(\theta + \frac{\tau }{4})=\cos(\theta )$

Note that sin(θ)=cos(θ) at

  • $\tau *\frac{1}{8}$

  • $\tau *\frac{5}{8}$

And that all these answers loop. That is, add any integer multiple of τ to θ and the results hold.

eiθ = eiθ + nτ

n ∈ ℕ

eiθ = cos(θ)+isin(θ)

eiθ = cos(θ + nτ)+isin(θ + nτ)

eiθ = ei(θ + nτ)

Tan

The tan(θ) function is defined as:

$\tan(\theta ):=\frac{\sin(\theta )}{\cos(\theta )}$

Behaviour around 0

sin(0)=0

cos(0)=1

$\tan(0):=\frac{\sin(0)}{\cos(0)}$

$\tan(0)=\frac{0}{1}$

tan(0)=0

Behaviour around cos(θ)=0

$\tan(\theta )=\frac{\sin(\theta )}{\cos(\theta )}$

So tan(θ) is undefined where cos(θ)=0.

This happens where:

$\theta=\frac{\tau }{4}+\frac{1}{2}n\tau$

$\theta=\frac{1}{4}\tau (1+2n)$

Where n ∈ ℤ.

Derivatives

$\tan(\theta )=\frac{\sin(\theta )}{\cos(\theta )}$

$\frac{\delta }{\delta \theta } \tan(\theta )=\frac{\delta }{\delta \theta }\frac{\sin(\theta )}{\cos(\theta )}$

$\frac{\delta }{\delta \theta } \tan(\theta )=\frac{\cos(\theta )}{\cos(\theta )}+ \frac{\sin^2(\theta )}{\cos^n(\theta )}$

$\frac{\delta }{\delta \theta } \tan(\theta )=1+\tan^2(\theta ) $

Cosine and sine

arccos(θ), arcsin(θ) and difficulty of inversing

In order to determine τ we need inverse functions for cos(θ) or sin(θ).

These are the arccos(θ) and arcsin(θ) functions respectively.

However this is not easily calculated. Instead we look for another function.

Calculating arctan(θ)

So we want a function to inverse this. This is the arctan(θ) function.

If y = tan(θ), then:

θ = arctan(y)

We know the derivative for tan(θ) is:

$\frac{\delta }{\delta \theta }\tan (\theta )=1+\tan^2(\theta )$

$\frac{\delta y}{\delta \theta }=1+y^2$

So

$\frac{\delta \theta }{\delta y}=\frac{1}{1+y^2}$

$\frac{\delta }{\delta y}\arctan (y)=\frac{1}{1+y^2}$

So the value for arctan(k) is:

$\arctan (k)=\arctan (a)+\int_a^k\frac{\delta }{\delta y}\arctan (y) \delta y$

$\arctan (k)=\arctan (a)+\int_a^k\frac{1}{1+y^2} \delta y$

What do we know about this function? We know it can map to multiple values of θ because the underlying sin(θ) and cos(θ) functions also loop.

We know that one of the results for arctan(0) is 0.
