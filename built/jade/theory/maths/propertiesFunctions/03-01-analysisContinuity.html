Continous functions

A function is continuous if:

limx → cf(x)=f(c)

For example a function $\frac{1}{x}$ is not continuous as the limit towards 0 is negative infinity. A function like y = x is continous.

More strictly, for any ϵ > 0 there exists

δ > 0

c − δ < x < c + δ

Such that

f(c)−ϵ < f(x)<f(c)+ϵ

This means that our function is continuous at our limit c, if for any tiny range around f(c), that is f(c)−ϵ and f(c)+ϵ, there is a range around c, that is c − δ and c + δ such that all the value of f(x) at all of these points is within the other range.
