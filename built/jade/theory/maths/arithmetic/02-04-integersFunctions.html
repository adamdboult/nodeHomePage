Functions of integers

Addition

Then we can define addition as:

(a, b)+(c, d)=(a + c, b + d)

Integer addition can then be defined:

a + b = {{a1},{a1, a2}} + {{b1},{b1, b2}}

a + b = {{a1 + b1},{a1 + b1, a2 + b2}}

Or:

a + b = c

c1 = a1 + b1

c2 = a2 + b2

Multiplication

Similarly, multiplication can be defined as:

(a, b).(c, d)=(ac + bd, ad + bc)

ab = c

c1 = a1b1 + a2b2

c2 = a2b1 + a1b2

Subtraction

a − b = c

c1 = a1 + b2

c2 = a2 + b1
