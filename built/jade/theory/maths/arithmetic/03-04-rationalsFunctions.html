0.1 Functions of rational numbers

0.1.1 Rational addition

Then we can define addition as:

(a, b)+(c, d)=(a.d + b.c, b.d)

a + b = c

c1 = a1b2 + a2b1

(a, b).(c, d)=(a.c, b.d)

ab = c
