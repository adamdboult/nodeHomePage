Neighbourhood topology

We have a set X.

For each element x ∈ X, there is a non-empty set of neighbourhoods N ∈ N(x) where x ∈ N ⊆ X such that:

  • If N is a subset of M, M is a neighbourhood.

  • The intersection of two neighbourhoods of x is a neighbourhood of x.

  • N is a neighbourhood for each point in some M ⊆ N
